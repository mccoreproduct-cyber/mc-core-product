

------------------------------------------------------------
🚀 Jak opublikować stronę i przyspieszyć indeksację Google
------------------------------------------------------------

1) Wgranie na GitHub Pages
- Wejdź do repo: mccoreproduct-cyber/mc-core-product
- Add file → Upload files → wgraj WSZYSTKIE pliki z paczki (rozpakowane) do katalogu głównego (tam gdzie index.html)
- Commit changes
- Settings → Pages → Deploy from branch: main / (root)

2) Szybka kontrola
- Główna strona: https://mccoreproduct-cyber.github.io/mc-core-product/
- Plik weryfikacyjny: https://mccoreproduct-cyber.github.io/mc-core-product/googlea4d8fd3e0e3ce427.html

3) Google Search Console (weryfikacja)
- Otwórz: https://search.google.com/search-console/
- Dodaj własność: Prefix URL → https://mccoreproduct-cyber.github.io/mc-core-product/
- Wybierz: Plik HTML → Zweryfikuj

4) Zgłoszenie do indeksacji
- W Search Console: Sprawdź URL → „Poproś o zaindeksowanie” dla strony głównej
- Powtórz dla kluczowych sekcji (/#about, /#cases, /#services)

5) Dodatkowe SEO
- robots.txt pozwala na indeksację (sprawdź w razie potrzeby)
- Wyślij sitemapę (sitemap.xml) w Search Console
- Dodaj link do strony w bio na LinkedIn oraz w innych miejscach (backlinki)
